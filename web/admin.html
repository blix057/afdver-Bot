üìã Bot Setup Instructions

  1. Deploy Vercel App: Deploy this app to Vercel with MongoDB credentials
  2. Register Bot: Use the form above to create a unique token for each bot instance
  3. Configure Bot: Set the bot's .env file:
    INGEST_URL=https://your-vercel-app.vercel.app/api/ingest
    INGEST_TOKEN=your_generated_token_here
  4. Update Vercel Environment: Add the new token to VALID_TOKENS in Vercel dashboard (comma-separated)
  5. Run Bot: The bot will now upload all collected links to your centralized system

üìù Required Vercel Environment Variables

# Required for the system to work:
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/?retryWrites=true&w=majority
MONGODB_DB=afdver_bot
MONGODB_COLLECTION=links
ADMIN_SECRET=your_secret_admin_token_here
VALID_TOKENS=bot1.token1,bot2.token2,bot3.token3

Note: Update VALID_TOKENS each time you register a new bot by adding the token to the comma-separated list.
